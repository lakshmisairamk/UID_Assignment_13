# IMDb Website Clone

## Project Overview
This project is a front-end clone of the IMDb (Internet Movie Database) homepage developed using HTML, CSS, and JavaScript. It replicates the look and feel of the IMDb website, including movie listings, celebrity sections, trailers, featured content, navigation menu, and responsive design.

## Features
- IMDb-style navigation bar
- Search bar with category selection
- Hero banner with automatic movie slider
- Up Next trailer section
- Featured Today section
- Most Popular Celebrities section
- What to Watch recommendations
- Top 10 Movies of the Week
- Fan Favorites section
- Popular Interests categories
- In Theaters section
- Top Box Office section
- Coming Soon trailers
- Streaming platforms section
- Born Today celebrities
- Top News section
- Recently Viewed movies section
- Responsive design for desktop and mobile devices
- IMDb favicon support

## Technologies Used

### HTML5
Used for creating the structure and content of the webpage.

### CSS3
Used for styling, layouts, animations, responsiveness, and visual appearance.

### JavaScript
Used for:
- Dynamic content generation
- Movie carousel functionality
- Automatic hero image slider
- Interactive buttons
- Menu toggle functionality
- Sign-in popup handling
- Scrolling controls
